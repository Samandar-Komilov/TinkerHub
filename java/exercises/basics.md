### Java Basics (Syntax Oriented)**. 

Scope is primitives, variables, arrays, control flow, methods. No OOP. No libraries beyond `java.lang`.

### Data Types & Variables

1. Declare variables of all primitive types. Assign sample values. Print each.
2. Read two integers. Print their sum, difference, product, and integer division result.
3. Convert an `int` to `double` and a `double` to `int`. Print both results.
4. Swap two integers using a temporary variable.
5. Swap two integers without using a temporary variable.
6. Given seconds as `int`, convert to hours, minutes, seconds.
7. Given a `char`, print its ASCII value.
8. Given an `int`, check whether it fits in `byte` and `short`.

---

### Branching: if / else

9. Given an integer, print whether it is positive, negative, or zero.
10. Given an integer, check if it is even or odd.
11. Given three integers, print the largest.
12. Given a year, check if it is a leap year.
13. Given a character, check if it is a vowel or consonant.
14. Given a score (0–100), print grade A/B/C/D/F using `if-else`.
15. Given temperature in Celsius, print a category: freezing, cold, warm, hot.

---

### switch

16. Given a number 1–7, print the weekday name.
17. Given a month number, print number of days (ignore leap years).
18. Simple calculator: given two integers and an operator (`+ - * /`), compute result.
19. Given a character grade (`A`, `B`, `C`, `D`, `F`), print a description.
20. Given an integer 1–4, print a season name.

### Arrays

21. Declare an integer array of size 10. Fill it with values 1–10. Print all elements.
22. Given an integer array, compute and print the sum of elements.
23. Find the maximum and minimum element in an array.
24. Count how many even and odd numbers are in an array.
25. Reverse an array in place.
26. Given an array, check whether a value exists in it.
27. Copy one array into another manually (no utility methods).

### Methods (Functions)

28. Write a method `isPrime(int n)` returning `boolean`. Test it.
29. Write a method `factorial(int n)` returning `long`. Handle `n = 0`.
30. Write a method that takes an integer array and returns the average as `double`.
