# Electronics - Learn by Doing

Progressive electronics projects: bare-bones circuits first, then Arduino.

## Philosophy
No formula memorization. Build it, measure it, break it, understand it.

## Components Available
- Breadboard (830-point)
- Jumper wires
- Resistors: 220R, 1K, 10K
- Transistor: BC547 (NPN)
- Digital multimeter (manual ranging)
- 9V batteries + snap connector
- LEDs (various colors)
- 10K potentiometer
- Arduino Uno kit

## Path

### 0_bare_bones - No microcontroller, just circuits
Build raw circuits with battery, resistors, LEDs, transistor, potentiometer.
Goal: confident with breadboard, multimeter, and core components.

### 1_arduino_uno - Microcontroller projects
Use Arduino to control circuits with code.
Goal: bridge software skills to hardware confidently.
